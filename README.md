# LAB-1-React-Creation-d-une-application-React-simple
# TP React — Composants & Props

TP d'initiation à React réalisé avec Create React App.

## Objectifs
- Créer des composants fonctionnels avec JSX
- Utiliser les props et PropTypes
- Structurer une application React

## Composants
| Fichier | Rôle |
|---|---|
| `Bonjour.js` | Message de bienvenue |
| `Salutation.js` | Salutation personnalisée avec un prénom |
| `Profil.js` | Affichage photo + nom utilisateur |
| `Car.js` | Informations d'une voiture |
| `ShoppingList.js` | Liste dynamique d'éléments |

## Lancement

```bash
npm install
npm start
```
